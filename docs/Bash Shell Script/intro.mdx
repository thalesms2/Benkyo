# Intro
> extenção `.sh`
## Shebang (sha-bang | hashbang)
> #!
Serve para identificar o interpretador de comandos.
No caso de um bash `#!/bin/bash`

## Linux terminal
```bash
vi nome.sh
```
Tecla `i`
```bash
#!/bin/bash
```
Digite o código do script
Tecla `ESC`
Digite `:wq`
Tecla `ENTER`
```bash
chmod a+x nome.sh
./nome.sh
```
