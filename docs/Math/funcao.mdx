## Dominio

Conjunto de dominio é todo elemento que pode substituir a variavel e resultar no Y.
Dominio da função é todo o que a função pode receber para substituir a varivel, isso sendo definido por possíveis restrições aritméticas.
Pode se caracterizar também, se pensado em um gráfico, os valores que estão na reta, pois é toda possíbilidade de valor que pode preencher 
a função e resolve-lá corretamente

Dominio = D(f)

## Função Constante

Para qualquer valor de X o valor de Y é sempre o mesmo
$f(x) = k$ o valor de Y vai ser o mesmo de X

Exemplo f(x) = a
Dominio D(f) = $\Reals$
Imagem Im(f) = {a}

O que informa que a função pode receber qualquer número que esteja no conjunto de números reais, e que a imagem ou seja o retorno sempre será a 
constante `a`.

## Igualdade de funções

Duas funções só são iguais se possuem igualdade de dominio

f(x) = x + 1 $\>\>\>$ $\rightarrow$ $\>\>\>$ D(f) = $\Reals$

g(x) = $\cfrac{(x - 1)(x + 1)}{x - 1}$ $\>\>\>$ $\rightarrow$ $\>\>\>$ D(g) = $\Reals \>$ - {1}

y = $\cfrac{\xcancel{(x - 1)}(x + 1)}{\xcancel{x - 1}}$ $\>\>\>$
y = x + 1

Mesmo que após cortar uma parte da função e a função `g` ficar igual a função `f`, elas não são iguais pois seus dominios são diferentes, dominio esse que
é validado da primeira formula da função.

## Função Par e Impar

Função par é quando o sinal de x não influencia o resultado.

f(x) = $x^2$

f(-x) = (-$x^2$) = (-x) * (-x) = $x^2$ = f(x)

Função impar é quando o sinal influencia no resultado final

g(x) = $x^3$

g(-x) = (-$x^3$) = (-x) * (-x) * (-x) = -$x^3$ $\ne$ f(x)
