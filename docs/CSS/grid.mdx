# Grid

```css
#grid {
    display: grid;
    grid-template-columns: 20% 20% 20% 20% 20%;
    grid-template-rows: 20% 20% 20% 20% 20%;
}

#id {
    grid-column-start: 2;
    grid-column-end: 6;
    grid-row-start: 2;
}

#id2 {
    grid-column: 2 / 6;
    grid-row: 2;
}
#id3 {
    grid-area: 1 / 2 / 4 / 6; /* grid-row-start / grid-column-start / grid-row-end / grid-column-end  */
}
```
